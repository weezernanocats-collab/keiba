## 1. データ収集 (Data Collection)

### 1.1 データソース

本システムのデータは全て **netkeiba.com** からスクレイピングで取得する。

| データ種別 | 取得元 | 取得タイミング | 格納先テーブル |

| レース一覧 | `race.netkeiba.com/top/race_list_sub.html` | 朝 09:00 JST | `races` |
| 出馬表（馬・騎手・枠順） | `race.netkeiba.com/race/shutuba.html` | レース一覧取得後 | `race_entries` |

| レース結果（着順・タイム・ラップ） | `race.netkeiba.com/race/result.html` | 17:00 全レース終了後 | `race_entries`, `races` |
| 追い切り評価 | `race.netkeiba.com/race/oikiri.html` | バッチ取得 | `model/oikiri_data.json` |

## 2. 予想エンジン (Prediction Engine)

### 2.1 二系統の予想パイプライン

本システムは**2つの独立した予測系統**を持ち、それぞれ異なる目的で使用する。

| 系統 | モデル | 特徴量 | 用途 |
|---|---|---|---|
| **ブレンド予想** | XGBoost + CatBoost アンサンブル | 53次元（オッズ含む） | Top 6推奨馬ランキング |
| **AI独自予想** | CatBoost No-Odds ランカー (v14.0) | 42次元（オッズ除外） | AI推奨買い目（単勝/複勝/馬連/ワイド） |

ブレンド予想は市場情報を取り込んだ総合的な順位予測。AI独自予想は市場に依存しない純粋なAI評価であり、市場が見落としている馬を発見する目的で設計されている。

### 2.2 17ファクタースコアリング

各出走馬に対して、以下の17要素を0〜100点で独立にスコアリングする。

#### 個体分析ファクター (10要素、合計80%)

| ファクター | 重み | 算出ロジック |
|---|---|---|
| **直近成績** | 17% | 直近5走の着順を加重平均 + グレード補正 + トレンド検出 + 馬体重トレンドボーナス |
| **距離適性** | 11% | レース距離 ±400m 以内の過去成績。距離が近いほど高評価 |
| **スピード指数** | 11% | 上がり3F + 走破タイムから算出。DB実データから動的基準タイム算出 |
| **騎手能力** | 8% | 通算成績70% + 直近30日フォーム30%のブレンド |
| **上がり3F** | 8% | 直近の上がり3Fタイムの平均。35秒以下で高評価 |
| **脚質適性** | 6% | 通過順位から判定（逃げ/先行/差し/追込）。距離によるボーナス |
| **馬場適性** | 5% | 芝/ダート、良/稍重/重/不良での過去成績 |
| **枠順分析** | 5% | 実データベースの枠別勝率、データ不足時は固定biasMapフォールバック |
| **安定性** | 5% | 着順の標準偏差が小さいほど高評価 |
| **ローテーション** | 4% | 前走からの間隔。最適間隔（3〜8週）で高得点 + 叩き良化ボーナス |

#### 統計ベースファクター (4要素、合計14%)

| ファクター | 重み | 算出ロジック |
|---|---|---|
| **血統適性** | 6% | 父馬産駒の芝/ダート・距離帯・馬場状態別成績 |
| **調教師能力** | 5% | 調教師の勝率・トラック別・距離カテゴリ別・馬場状態別・グレード別 |
| **季節パターン** | 2% | 馬ごとの月別成績パターン |
| **斤量アドバンテージ** | 1% | 平均斤量との差分 |

#### 市場シグナルファクター (3要素、合計6%)

| ファクター | 重み | 算出ロジック |
|---|---|---|
| **市場オッズ** | 3% | 単勝オッズの逆数を対数正規化 |
| **天候適性** | 2% | 当該天候条件での過去成績 |
| **着差競争力** | 1% | 僅差好走の頻度 |

### 2.3 カテゴリ別ウェイト + 競馬場別補正

| カテゴリ | 条件 | 主な重み調整 |
|---|---|---|
| **turf_sprint** | 芝 ≤1400m | スピード指数×1.3、枠順×1.5 |
| **turf_mile** | 芝 1401-1800m | ほぼ基本重み（基準） |
| **turf_long** | 芝 1801m+ | 血統×1.3、脚質×1.3 |
| **dirt_sprint** | ダート ≤1400m | スピード×1.4、枠順×1.3 |
| **dirt_long** | ダート 1401m+ | 馬場適性×1.3、スピード×1.2 |

10競馬場別に補正（東京: 上がり3F×1.4, 中山: 枠順×1.5 等）。

### 2.4 当日馬場バイアス補正

`track-bias.ts` で同日・同競馬場の結果確定済みレース（3R以上）を分析。

| バイアス | 調整幅 |
|---|---|
| 枠順バイアス（内枠/外枠有利度） | ±2点 |
| 脚質バイアス（先行/差し追込有利度） | ±2点 |
| 騎手当日フォーム（好調/不調） | ±1.8点 |

信頼度（3R=0.375 〜 8R+=1.0）でスケール。予想詳細ページから手動で再計算可能（`?biasUpdate=1`）。

## 3. MLモデル

### 3.1 XGBoost + CatBoost アンサンブル（ブレンド予想用）

53次元特徴量を使用。オッズ・人気を含む全情報を活用した総合ランキング。

**推論フロー（TypeScriptネイティブ）:**
```
buildMLFeatures() → 53次元ベクトル
  │
  ├─ [優先] カテゴリ別モデル (5カテゴリ)
  │    ├─ xgb_ranker_{cat}.json → softmax → Isotonic較正
  │    ├─ catboost_ranker_{cat}.json → softmax → Isotonic較正
  │    └─ アンサンブル: XGB×w + CB×w (ensemble_weights.json)
  │
  ├─ [フォールバック1] グローバルモデル
  └─ [フォールバック2] 17因子スコアからsoftmax確率推定
```

Python不要 — XGBoost/CatBoostのJSON決定木をTypeScriptで直接パースし推論。

**アンサンブル重み (現在 — 2026-04-13再学習):**

| カテゴリ | XGBoost | CatBoost |
|---|---|---|
| グローバル | 0.0 | 1.0 |
| turf_sprint | 0.05 | 0.95 |
| turf_mile | 0.25 | 0.75 |
| turf_long | 0.0 | 1.0 |
| dirt_short | 0.0 | 1.0 |
| dirt_long | 0.2 | 0.8 |

**精度 (ranker_v6 — 2026-04-13再学習):**

| 指標 | XGBoost Val | CatBoost Val | Test |
|---|---|---|---|
| NDCG@1 | 0.4952 | 0.5057 | 0.4795 |
| NDCG@3 | 0.5513 | 0.5581 | 0.5320 |
| Top-1精度 | 31.7% | 32.3% | 29.6% |
| Top-3精度 | 68.2% | 68.1% | 62.0% |

### 3.2 CatBoost No-Odds ランカー v14.0（AI独自予想用）

市場情報を完全に排除した42次元特徴量で学習。AI推奨買い目の生成に使用。

**学習設定:**
- 損失関数: PairLogitPairwise
- depth=6, learning_rate=0.02, iterations=547
- 4段階ラベル（1着=3, 2着=2, 3着=1, 他=0）

**42特徴量 (feature_names_no_odds.json):**
```
jockeyAbility, trainerDistCatWinRate, sireTrackWinRate, grade_encoded,
age, jockeyTrainerWinRate, relativePosition, lastRacePosition,
```

## 4. AI推奨買い目

### 4.1 概要

No-Oddsモデルの予測確率（noOddsProbs）から、市場に依存しない買い目を自動生成する。`generateAIRankingBets()` で実装。

データ分析（AI単勝ROI 179.5%、複勝率59.3%）に基づき、単勝・複勝を中心に据え、馬連・ワイドはパターンに応じて補助的に推奨する。

### 4.2 Gap分析によるパターン判定

上位馬間の確率差（Gap）を分析し、4パターンに分類:

| パターン | 条件 | 買い目 |
|---|---|---|
| **一強** | gap12 > 8% | 単勝(1位) + 複勝(1位) + 馬連(1-2位)（3点） |
| **二強** | gap23 > 5% | 複勝(1-2位) + 馬連(1-2位)（2点） |
| **三つ巴** | gap34 > 3% | 複勝(1位) + ワイド(1-2位)（2点） |
| **混戦** | それ以外 | 複勝(1位)（1点） or 見送り |

### 4.3 しょーさん予想（v15.0）

先行力 × 休養 × アゲ騎手の乗り替わりに基づくルールベース予想。`src/lib/shoshan-theory.ts`。

**前提条件:**
- 直近10走以内のJRA（中央競馬）レースで1角1-2番手の経験あり
- 地方競馬での先行実績はカウントしない

**理論1（前走凡走+アゲ騎手乗替・復調パターン）:**
- 必須: 乗り替わり（アゲ騎手への交代）
- 必須: 前走4着以下
- 必須: 先行力3回以上（直近10走JRAレース、1角1-2番手）
- スコアリング: 先行力回数(3=+5, 4=+10, 5+=+15) + ゾーンボーナス(Z2=+20, Z1=+15, Z3=+10, Z4=+5)
- 休養ボーナスは削除（バックテストで42-90日がROI 87.8%赤字のため）
- バックテスト: 641件, **単勝ROI 140.6%**

**理論2（前走好走+乗替・好調継続パターン）:**
- 必須: 乗り替わり（継続騎乗は除外）
- 必須: 前走3着以内
- 必須: 前走の騎手がZone 1-2 or ルメール・川田（一流騎手から乗り替わり）
- 今走の騎手がアゲ騎手（Zone 1-4）
- バックテスト: 勝率24.5%, **単勝ROI 126%**

**アゲ騎手リスト:**
- Zone 1: 武豊・松山・横山武・坂井（長年好成績、過剰人気気味）
- Zone 2: 岩田望・鮫島駿・荻野極（直近好調、最高期待値）
- Zone 3: 田辺・菱田・西塚・石川・浜中・三浦・斎藤・田山
- Zone 4: 丹内・佐々木（一時期Zone2、現在期待値低）
- Zone 5: 丸山（地方競馬場のみZone3扱い）
- ※ ルメール・川田は過剰人気のため**意図的に除外**

**マッチスコア:** 0-100%。最低45点で候補入り。Top 3頭を表示。

**しょーさん × AI 掛け合わせ表示:**
- 🔥 **スイートスポット**: しょーさん65%+ ∩ AI Top3 ∩ オッズ5倍以上（バックテストROI 116%）
- 💎 **AI見落とし穴馬**: しょーさん×AI Top3外（バックテストROI 107%）
- ⚠️ **AI同意人気**: しょーさん∩AI Top3（過剰人気気味、参考表示）

### 4.4 パドック解説リアルタイム監視（v15.0）

グリーンチャンネルのYouTube Live配信から音声を取得、Whisperで文字起こし、ローカルLLMで要約して予想ページに表示。

**パイプライン:** `scripts/paddock-watcher.sh`
1. yt-dlp + ffmpegで60秒チャンクでライブ音声を取得
2. Whisper(tiny)で日本語文字起こし（60秒音声を約7秒で処理）
3. 競馬場名追跡 + 馬名マッチングで該当レースを判定
4. パドックキーワード（馬体、歩き、気配、仕上がり等）でフィルタ
5. Ollama(gemma3:4b)で馬番ごとの簡潔な評価に要約
6. 発走7分前に自動で予想再生成、Slack DM通知

**Slack連携:**
- 開催日朝にYouTube URL送信リマインダー（土日 8:30 JST）
- ユーザーが Slack DM に YouTube URL を送信 → `paddock-launcher.sh` が検知して watcher 自動起動
- 予想再生成完了時に Slack DM 通知

- 予想一覧ページ: 各レース行にバッジ表示。「買い推奨のみ」フィルタで絞り込み可能

## 5. 市場オッズブレンド + Value Betting

### 5.1 オッズブレンド

```
市場暗示確率 = (1/odds) / Σ(1/odds)   (オーバーラウンド除去)
ブレンド確率 = log-odds空間でモデル × (1-w) + 市場 × w
乖離度 = modelProb - marketProb   (正=妙味あり)
```

### 5.2 Kelly Criterion

```
f* = (b×p - q) / b
推奨: f*/4 (Fractional Kelly)
MAX_STAKE: 25%
```

Filter E（バックテスト検証済み）: ダsprint除外 & オッズ3-50倍 & 乖離>3% → ROI 243.1%
