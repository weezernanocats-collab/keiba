# デプロイ前チェック

本番デプロイ前に必要な全チェックを実行します（2026-03-21障害の再発防止）。

## チェック項目

### 1. ビルドチェック
```bash
npm run build
```

### 2. テスト実行
```bash
npm test
```

### 3. Lint
```bash
npm run lint
```

### 4. クエリ変更の確認
- `git diff` でSQL変更箇所を特定
- 相関サブクエリ `WHERE x = (SELECT ...)` がないか確認
- JOINでの行数爆発リスクチェック

### 5. データ安全性
- DELETE文にWHERE句があるか
- libsql:// が混入していないか
- db.batch() が使われていないか

### 6. 影響範囲の明示
- 変更ファイル一覧
- 影響するAPIエンドポイント
- 影響するテーブル
- 「影響ゼロ」は禁句 — 具体的に列挙する

## 出力フォーマット

各項目について ✅ / ❌ で結果を表示し、❌がある場合は修正案を提示する。
