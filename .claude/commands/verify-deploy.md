# デプロイ後検証

Vercelデプロイ後にAPIエンドポイントの動作を自動確認します。

## チェック対象

引数でベースURLを指定（デフォルト: https://keiba-five.vercel.app）

### 必須エンドポイント
1. `GET /api/races?date=今日の日付` → 200 + レース一覧が返ること
2. `GET /api/predictions/{最初のraceId}` → 200 + 予測データが返ること
3. `GET /api/stats` → 200 + 統計データが返ること
4. `GET /api/diagnostic` → 200 + DB接続OKであること

### 確認内容
- HTTPステータスコード
- レスポンスタイム（5秒以内）
- レスポンスボディが空でないこと
- JSON形式が正しいこと

## 出力

各エンドポイントについて ✅ / ❌ + レスポンスタイムを表示
